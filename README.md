## ⚗️ Installation

Clone the project

```bash
  git clone https://github.com/rupam-seal/taskify.git
```

<br>

Go to the project directory

```bash
  cd taskify

```

<br>

Install Dependencies

```bash
  npm install

```

<br>

Run server

```bash
  npm run dev
```
